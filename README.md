# StayConnect 🎓🏠

**Find Home. Find Your People.**

StayConnect is a student-centric hostel discovery and roommate matching platform designed specifically for university students in Pakistan.

🌐 **Live Demo:** [https://muhammad-sufiyan-pk.github.io/Stay-Connect/](https://muhammad-sufiyan-pk.github.io/Stay-Connect/)

---

## Features

- 🏠 **Hostel Discovery** — Browse verified hostels across Pakistan with real photos, pricing, amenities, and reviews
- 👥 **Roommate Matching** — AI-powered compatibility matching based on lifestyle, budget, interests, and study habits
- 💬 **Secure Messaging** — In-app chat with real-time messaging between students
- 🛡️ **Student Verification** — CNIC and university ID verification for a trusted community
- 🏘️ **Student Communities** — Join city-based and university-based student groups
- ❤️ **Save & Compare** — Save hostels and roommate profiles for easy comparison
- 📱 **Mobile-First Design** — Premium UI optimized for mobile devices

## Pakistani Localization

- 🇵🇰 Pakistani universities (LUMS, IBA, NUST, FAST, COMSATS, etc.)
- 💰 Pricing in Pakistani Rupees (Rs.)
- 🏙️ Major cities covered (Karachi, Lahore, Islamabad, Rawalpindi, etc.)
- 👤 Authentic Pakistani student profiles

## Tech Stack

- **React 18** (loaded via CDN)
- **Vanilla CSS** with CSS variables
- **Single-page application** — no build step required
- **GitHub Pages** deployment
